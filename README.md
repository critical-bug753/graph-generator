# 📊 Graph Generator
**An App by Critical Bug**

Graph Generator is a native Android application built with **Kotlin** and **Jetpack Compose**. It provides a clean interface for users to input numerical data pairs, visualize them through dynamic line graphs, and export the results as printable PDF documents.



## ✨ Features
* **Real-time Visualization:** Input X and Y coordinates to see your graph update instantly.
* **Smart Axis Scaling:** Axes automatically adjust to fit the range of your data points perfectly.
* **PDF Generation:** Export your work into a professional A4-sized PDF report including both the graph and the raw data table.
* **Modern Material 3 UI:** A clean, intuitive design featuring smooth transitions and a custom splash animation.

## 🛠 Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Graphing:** MPAndroidChart
* **PDF Engine:** Android Print/PdfDocument API

## 🚀 Getting Started
1. Clone this repository.
2. Open in **Android Studio**.
3. Sync Gradle and run on an emulator or physical device.
